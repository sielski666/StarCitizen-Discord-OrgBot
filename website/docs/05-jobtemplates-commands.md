# 06 - Event Template Commands

Templates are now event-focused and used with `/eventjob post`.

## `/eventtemplate add`
**Who:** admin

Opens modal to create an event template.

## `/eventtemplate update name:<template>`
**Who:** admin

Opens prefilled modal for event template updates.

## `/eventtemplate clone source_name:<src> new_name:<dst>`
**Who:** admin

Copies one event template to a new name.

## `/eventtemplate list [include_inactive:true|false]`
Lists event templates.

## `/eventtemplate view name:<template>`
Shows full details for one event template.

## `/eventtemplate disable name:<template>`
Disables template.

## `/eventtemplate enable name:<template>`
Enables template.

## `/eventtemplate delete name:<template>`
Deletes template.

## `/eventjob post template:<name>`
Posts an event job from an event template.
