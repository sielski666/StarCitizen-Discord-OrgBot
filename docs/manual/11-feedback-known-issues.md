# 11 - Feedback & Known Issues

## Feedback should target bot behavior
Use this format:
- command/action
- job ID / event ID
- expected behavior
- actual behavior
- screenshot/time (UTC)

## Suggested categories
- setup
- template UX
- event attendance sync
- payout/finance
- permissions
- reliability/performance

## Known caveats
- live Discord permissions affect event behavior
- scheduled event availability impacts sync outcomes
